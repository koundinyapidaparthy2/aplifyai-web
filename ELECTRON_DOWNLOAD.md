# AplifyAI Desktop App

Download the AplifyAI desktop application for offline access and enhanced performance.

## macOS

### Download
- [Download AplifyAI for macOS (Apple Silicon)](https://github.com/koundinyapidaparthy2/jobseek-chromeextension/releases/download/v1.0.0/AplifyAI-1.0.0-arm64.dmg) - M1/M2/M3 Macs
- [Download AplifyAI for macOS (Intel)](https://github.com/koundinyapidaparthy2/jobseek-chromeextension/releases/download/v1.0.0/AplifyAI-1.0.0-x64.dmg) - Intel Macs

### Installation
1. Download the appropriate DMG file for your Mac
2. Open the DMG file
3. Drag AplifyAI to your Applications folder
4. Launch AplifyAI from your Applications folder

**Note:** macOS may show a security warning for unsigned apps. To open:
1. Right-click the app and select "Open"
2. Click "Open" in the security dialog

## Building from Source

If you prefer to build the desktop app yourself:

```bash
cd aplifyai-web
npm run electron:build
```

The built application will be in the `dist` directory.

## Requirements
- macOS 11.0 or later (Big Sur+)
- 4GB RAM minimum
- Internet connection for AI features

## Features
- Full offline access to your resumes and cover letters
- Faster performance compared to web app
- Native macOS integration
- System tray support
- Auto-updates (coming soon)
